# LAI AI - Joseph's Assistant

## Objective (English)

**LAI AI (Leoliver's Assistant Intelligence)** is a premium, culturally-aware AI chatbot designed to serve the Lai Hakha-speaking community with warmth, wisdom, and care. Built with cutting-edge technology (Next.js, Google Gemini), LAI AI preserves and celebrates Lai language and culture while providing modern AI assistance.

### Core Mission:
- **Cultural Preservation**: Maintain pure Hakha Lai language with proper tone markers and grammar
- **Emotional Intelligence**: Respond with deep empathy, care, and cultural sensitivity
- **Community Connection**: Bridge traditional values with modern technology
- **Accessibility**: Make AI technology accessible to Lai speakers worldwide

### Key Features:
- Pure Hakha Lai language support with perfect tone marks (á, é, í, ó, ú, ṭ)
- Warm, caring, and wise conversational tone inspired by "Ram King" style
- Bilingual support (Lai Hakha & English)
- File upload and image analysis (Gemini Vision API)
- Persistent chat history with Supabase
- Beautiful, responsive UI with dark/light themes
- Social authentication (Google, GitHub)

## Tum Duhnak (Lai Holh)

**LAI AI (Leoliver's Assistant Intelligence)** cu Lai Hakha holh hman mi zatlangbu caah a sak mi AI chatbot a si. Lungthin a thiang, mifim, le dawtmi tein biaruahnak a pe. Technology thar (Next.js, Google Gemini) hmang in Lai holh le nunphung kan humhim le kan upat.

### Tum Duhnak Ṭha Bik:
- **Holh Humhimnak**: Hakha Lai holh dik le fiang tein tone marker kip in hman
- **Lungthin Hngalhnak**: Siaherhnak, dawtnak, le nunphung zulhphung in leh
- **Zatlangbu Pehtlaihnak**: Nunphung hlui le technology thar pehtlaih
- **Hmuh Theihnak**: Vawleicung Lai holh hman mi kip caah AI technology pe

### Thil Tuah Theihnak Ṭha Bik:
- Hakha Lai holh dik tein tone marker (á, é, í, ó, ú, ṭ) kip in hman
- Lungthin a thiang, dawtmi, le mifim tein "Ram King" zulhphung in biaruah
- Holh pahnih (Lai Hakha le English) hman theih
- File upload le thlahmuh zirchiannak (Gemini Vision API)
- Biaruahnak hlui kan humhim (Supabase)
- UI a hmel le responsive tein dark/light theme
- Social authentication (Google, GitHub)

## Technology Stack

- **Frontend**: Next.js 14, React, TypeScript, Tailwind CSS
- **AI**: Google Gemini 2.5 Flash with Vision API
- **Database**: Supabase (PostgreSQL)
- **Authentication**: Supabase Auth (Google, GitHub OAuth)
- **Hosting**: Vercel (recommended)
